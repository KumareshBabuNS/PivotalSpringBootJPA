<h2> Steps to Run Locally </h2>

- Clone as follows

```
> git clone https://github.com/papicella/PivotalSpringBootJPA.git
```

- package as shown below

```
> mvn package
```

- Run as follows

```
mvn spring-boot:run
```

- Access as follows

```
http://localhost:8080/
```
